# Zealot

A personal learning project for exploring game development in Unity. The goal is to build hands-on experience with Unity 6, HDRP rendering, and 3D game systems from the ground up.

**Status:** Early-stage — HDRP environment set up, minimal game logic implemented so far.

## Tech

- **Engine:** Unity 6 (6000.3.11f1)
- **Render Pipeline:** HDRP 17.3.0
- **Platform:** Windows 64-bit (StandaloneWindows64)
- **Input:** Unity Input System
