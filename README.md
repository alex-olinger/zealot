# Zealot

A 3D game built with Unity 6 and the High Definition Render Pipeline (HDRP).

## Requirements

- Unity 6000.3.11f1
- Windows 10/11 64-bit

## Getting Started

1. Clone the repository
2. Open Unity Hub and add the project folder
3. Open the project with Unity 6000.3.11f1
4. Load `Assets/OutdoorsScene.unity`

The `Library/` folder will be regenerated on first open — this may take a few minutes.
